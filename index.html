Comandos GitHub (Básicos)

  1. Para acessar a pasta do repositório local.
    2. cd + Nome da pasta
  2. Setar Usuário
    3. git config --global user.name "Nome do Usuário"
  3. setar E-mail
    4. git config --global user.email "digite seu e-mail"
  4. Criando um repositório
    5. git init
  5. Verificar status
    6. git status
  6. Adicionar todos os arquivos/ diretórios
    7. git add .
  7. Comitar um arquivos
    8. git commit -m "Comentário ex. Atualizaçao..."
  8. Informar a pasta remota do github
    9. git remote add origin http:// (nesse momento copie e cole o endereço do github).
  9. Publicar as atualizações no repositório remoto
    10. git push -u origin master
  10. Criando um novo bransh
    11. git branch bug-123
  11. Criando um novo bransh
    12. git checkout -b bug-456
  12. Baixar as alterações no repositório remoto
    13. git pull
